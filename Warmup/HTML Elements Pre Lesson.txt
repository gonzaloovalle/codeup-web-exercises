# HTML Elements Pre Lesson
### Overview
1. In your own words, define an HTML element.
ELements are the parts of HTML that give a page structure. 

2. What is a void element?
A void element is an element that does not have any content and not closing tag.

3. What do all elements begin and end with
All elements begin with angle brackets.

4. What is the syntax of an HTML element
The element is enclosed with angle brackets. Most elements have a closing tag also enclosed with angle brackets. The opening and closing tags usually go around some type of content. 

5. What do we call an element that contains other elements? What are the nested elements called?
An element that contains other elements is called a parent. The nested elements are called children. 

### Block-Level vs. Inline Elements
1. What is a block level element? What is an example of a block level element?
A block level element takes up all of the space inside its parent element. An example of a block element is "<div></div>"

2. What is an inline element? What is an example of an inline element?
An inline element is an element that is bound only by the tags that define it within a line. "<span></span>" is an example of an inline element.

### Attributes
1. What is an attribute?
Attributes are additional data inside of the element tag.
